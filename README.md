# CallBlocker

Android向けの迷惑電話ブロックアプリケーションです。

## 概要

このアプリは、特定の番号やルールに基づいて着信をブロックし、ユーザーを迷惑電話から守ります。

## 主な機能

- **着信ブロック**: 指定した番号やパターンに基づき、自動的に着信を拒否します。
- **着信履歴管理**: ブロックした履歴を確認できます。
- **ルール設定**: 柔軟なブロックルールの作成が可能です。

## 技術スタック

- **言語**: Kotlin
- **プラットフォーム**: Android
- **開発環境**: Android Studio

## ビルド要件

- Android SDK
- JDK 17 (推奨)

## 注意事項

このプロジェクトのビルドには `local.properties` ファイルにAndroid SDKのパス設定が必要です。
Android Studioでプロジェクトを開くと自動的に生成される場合があります。
